## Full Description
**Data Widget** is a powerful and flexible tool that allows you to bring any remote JSON data directly to your Android home screen. Whether you are monitoring server statuses, tracking crypto prices, or counting down to a special event, Data Widget gives you the tools to build a professional dashboard in minutes.

**Key Features:**
*   **Flexible Grid System:** Build custom layouts using a Bootstrap-inspired grid (1 to 12 columns). Arrange your data exactly how you want it.
*   **Per-Second Precision:** Built-in support for live countdown timers and real-time updates that stay accurate to the second.
*   **Complete Visual Control:** Customize background colors, transparency, font sizes, text alignment, and corner radius to match your wallpaper perfectly.
*   **GET/POST Support:** Choose the HTTP method that fits your API requirements.
*   **Template Library:** Save your favorite configurations and quickly deploy them to new widgets.
*   **Battery Efficient:** Advanced optimization automatically pauses updates when the screen is off to save power and traffic.
*   **Developer Friendly:** Use the built-in Test tool to validate your JSON structure and connection instantly.

**How it works:**
1. Provide a URL to a JSON file hosted on your server or GitHub.
2. Define your rows and columns in the JSON (simple schema).
3. Customize the widget's appearance in the app.
4. Add the widget to your home screen and watch your data come to life!

**Privacy First:**
Data Widget stores all your URLs and settings locally on your device. We do not collect your data or track your usage.
